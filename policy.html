POLÍTICA DE PRIVACIDADE – BIPVIEW

Última atualização: 30/11/2025

A BIPVIEW (“BIPVIEW”, “nós”) leva sua privacidade a sério. Esta Política explica de forma transparente como coletamos, usamos, compartilhamos e protegemos seus dados pessoais quando você utiliza nossos aplicativos móveis, sites, APIs e demais serviços (conjuntamente, “Serviços”).

Ao criar uma conta, acessar ou usar os Serviços, você declara que leu, compreendeu e concorda com esta Política.

2. Escopo

Esta Política se aplica aos dados pessoais tratados pela BIPVIEW quando você:

  • cria conta e configura seu perfil;
  • conecta corretoras/exchanges, carteiras ou integrações via API;
  • monta e acompanha portfólios;
  • usa gráficos, listas, alertas e relatórios;
  • interage com suporte, comunidade ou conteúdos;
  • navega em nossos sites e páginas oficiais.

3. Dados que coletamos

Coletamos somente os dados necessários para prestação do serviço, melhoria contínua e segurança. Os dados podem ser fornecidos diretamente por você, gerados pelo uso dos Serviços ou recebidos via integrações autorizadas.

3.1 Dados fornecidos por você

  • Dados de cadastro: nome, e-mail, senha (criptografada), país/idioma, foto (opcional).
  • Preferências: ativos favoritos, watchlists, parâmetros de alertas, layout.
  • Conteúdo do usuário: anotações, comentários, relatórios personalizados, arquivos/imports (quando aplicável).
  • Comunicações: mensagens enviadas a suporte, feedbacks, respostas a pesquisas.

3.2 Dados coletados automaticamente

  • Dados de uso: páginas/telas acessadas, cliques, tempo de sessão, eventos do app, erros.
  • Dados do dispositivo: modelo, sistema operacional, versão do app, identificadores do dispositivo (ex.: Android Advertising ID/AAID quando permitido), idioma, fuso horário, IP aproximado.
  • Dados de diagnóstico: logs técnicos e relatórios de falha (crash reports).

3.3 Dados recebidos por integrações (com sua autorização)

Quando você conecta exchanges, corretoras, wallets ou APIs suportadas, podemos tratar:

  • Endereços públicos de carteira (quando você inserir/autorizar sincronização).
  • Saldos e posições.
  • Histórico de transações (compra, venda, depósito, saque, taxas).
  • Chaves de API e tokens de acesso somente com as permissões que você escolher.

Importante: a BIPVIEW não solicita nem armazena suas senhas de exchanges/corretoras. Chaves de API são armazenadas de forma segura e, quando possível, com escopo mínimo e criptografia forte.

3.4 Dados sensíveis

A BIPVIEW não tem como objetivo coletar dados sensíveis. Caso algum dado sensível seja tratado por necessidade técnica, isso será informado de forma destacada no app, com consentimento específico quando exigido.

4. Para que usamos seus dados (finalidades)

Usamos seus dados para:

1. Prestar os Serviços principais

  • criar e administrar sua conta;
  • exibir preços de criptoativos, gráficos e indicadores;
  • calcular desempenho, P&L, alocação e risco;
  • gerar relatórios e insights de portfólio;
  • disparar alertas configurados por você.

2. Manter segurança e integridade

  • autenticação, prevenção de fraude, abuso e acessos não autorizados;
  • auditoria e investigação de incidentes.

3. Melhorar a plataforma

  • entender uso e performance;
  • corrigir bugs;
  • desenvolver novas funcionalidades.

4. Comunicação e suporte

  • atender solicitações;
  • enviar avisos operacionais (ex.: mudanças de API, manutenção, segurança).

5. Marketing e relacionamento (quando aplicável)

  • envio de novidades, conteúdos e ofertas, com opt-out fácil;
  • não usamos dados de portfólio para publicidade de terceiros.

5. Bases legais (LGPD)

Tratamos dados com base em:

  • Execução de contrato: para operar conta, portfólio, relatórios e integrações.
  • Legítimo interesse: melhoria do produto, segurança, métricas internas, prevenção a fraudes.
  • Consentimento: quando você autoriza integrações específicas, notificações não essenciais ou tratamentos não esperados.
  • Cumprimento de obrigação legal/regulatória, quando aplicável.

6. Compartilhamento de dados

Compartilhamos dados apenas quando necessário e nos limites desta Política:

6.1 Prestadores de serviço / operadores

Podemos compartilhar dados com empresas que nos ajudam a operar a BIPVIEW, como:

  • infraestrutura cloud/hosting;
  • analytics e métricas;
  • crash/monitoramento de erros;
  • notificações push/e-mail;
  • processadores de pagamento/assinatura.

Esses terceiros atuam como operadores, seguindo nossas instruções e medidas de segurança compatíveis com esta Política.

6.2 Integrações escolhidas por você

Ao conectar uma exchange/corretora/wallet, você autoriza o tráfego de dados entre a BIPVIEW e o provedor escolhido, nos limites das permissões concedidas.

6.3 Requisições legais

Podemos divulgar dados para cumprir lei, ordem judicial ou regulatória, ou para proteger direitos da BIPVIEW, usuários e terceiros.
